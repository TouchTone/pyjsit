Justseed.it Uploader Configuration

This is a simple extension to upload hash/magnet/torrents to your justseed.it account. To do that you need to create an API key on your Options page and enter it here:

After that you just have to right-click on an info hash or a .torrent or magnet: link and select "Upload to JSIT" or
add buttons to torrent and magnet links.
